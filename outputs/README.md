# Outputs del Proyecto

Este folder reúne los entregables finales de análisis, reporting y visualización.

## Estructura rápida
- `dashboard/`: dashboard ejecutivo interactivo y punto de acceso principal.
- `charts/`: gráficos estáticos exportados para soporte analítico.
- `reports/`: reportes de validación, QA, hallazgos y release.
