# Supuestos de Modelado

1. El horizonte simulado cubre 90 días industriales con 3 turnos diarios.
2. El mix EV evoluciona de 25% a 60% en el periodo (ramp-up).
3. Solo EV requiere sesiones de carga para readiness de expedición.
4. La red de cargadores tiene capacidad limitada y sufre variación por disponibilidad energética.
5. La ocupación de patio superior a 85% aumenta no linealmente tiempos de espera.
6. En turnos nocturnos se reduce dotación y capacidad efectiva.
7. El cumplimiento SLA depende de secuencia de eventos y no solo de salida final.
8. Los datos son sintéticos, pero calibrados para patrones plausibles de automoción.
